# Role Permission Table Builder

**Category:** misc
**Status:** active
**Version:** v1

## What it does
Generates comprehensive role-based access control (RBAC) permission matrices in markdown or SQL format covering pages, UI components, data operations, and Server Actions, along with middleware protection code, component-level permission guards, and server action authorization utilities.

## When to use
- Designing an authorization system for a new application feature or project
- Documenting existing permissions in a structured, shareable format
- Generating SQL schema for a roles/permissions database structure
- Creating TypeScript middleware and component guards from a permission matrix

## What it produces
Produces markdown permission matrix tables (page, component, data, action levels), SQL schema for RBAC tables with seed data, Next.js middleware protection code, a ProtectedComponent wrapper, and server action permission guard utilities.
