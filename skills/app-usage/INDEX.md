| [SkillGuard ⭐](skillguard/) | Acts as a mandatory pre-install security gate that reviews any external file, skill, config, script, or package before it is installed or activated in the Claude workflow. | [Using Superpowers](using-superpowers/) | Establishes the meta-protocol for skill invocation: any task with even a 1% chance of skill applicability must invoke the Skill tool before taking any action, including before asking clarifying questions. Defines skill priority order and the rule that user instructions always override skill content. | v1 | ✅ active |
| [Writing Skills](writing-skills/) | Applies TDD principles to skill authoring: run baseline scenarios without the skill first (RED), write minimal skill content addressing the observed failures (GREEN), then close loopholes by finding new rationalizations and adding explicit counters (REFACTOR). Covers YAML frontmatter requirements, Claude Search Optimization, flowchart usage, and token efficiency. | v1 | ✅ active |
