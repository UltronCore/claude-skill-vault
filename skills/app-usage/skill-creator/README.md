# Skill Creator

**Category:** app-usage
**Status:** active
**Version:** v1
**Source:** skill-creator@claude-plugins-official

## What it does
Guides the full skill creation lifecycle: capturing intent, interviewing for edge cases, writing SKILL.md, generating test cases, running with-skill and baseline subagents in parallel, grading outputs quantitatively, launching an eval viewer for human review, iterating based on feedback, and optionally optimizing the description for better triggering accuracy.

## When to use
- When creating a new skill from scratch
- When improving or benchmarking an existing skill
- When optimizing a skill's description for triggering accuracy

## What it produces
A tested, iterated SKILL.md with quantitative benchmark data, human-reviewed outputs, and an optimized description — packaged as a .skill file when the present_files tool is available.
