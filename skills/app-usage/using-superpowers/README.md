# Using Superpowers

**Category:** app-usage
**Status:** active
**Version:** v1
**Source:** superpowers@claude-plugins-official

## What it does
Establishes the meta-protocol for skill invocation: any task with even a 1% chance of skill applicability must invoke the Skill tool before taking any action, including before asking clarifying questions. Defines skill priority order and the rule that user instructions always override skill content.

## When to use
- At the start of every conversation where superpowers skills are available
- Before any response when skills might apply to the task

## What it produces
Correct skill invocation behavior with proper announcement, TodoWrite creation for checklists, and exact skill content followed without deviation.

## Related skills
All superpowers skills — this is the entry point meta-skill
