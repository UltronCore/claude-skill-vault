# Agent Loop Patterns

**Category:** orchestration
**Status:** active
**Version:** v1

## What it does
Provides design patterns for building autonomous agent loops, task queues, and multi-agent crew systems with Claude.

## When to use
- Building self-directed agents that run without constant human input
- Implementing task decomposition and queue-based execution
- Designing role-based agent workflows (researcher, analyst, writer)
- Parallelizing independent subtasks across multiple agents

## What it produces
Working patterns for BabyAGI-style task loops, crewAI role-based crews, ReAct reasoning loops, and AutoGPT-style swarm dispatch.
