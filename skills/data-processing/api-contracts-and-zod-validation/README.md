# API Contracts and Zod Validation

**Category:** data-processing
**Status:** active
**Version:** v1

## What it does
Generates Zod schemas and TypeScript types for API routes, Server Actions, and forms. Automates the creation of runtime validation at client-server boundaries using TypeScript-to-Zod schema generation, pre-built templates, and a Python automation script.

## When to use
- When adding input validation to an API route or Next.js Server Action
- When creating type-safe form schemas with React Hook Form or similar
- When establishing an API contract between client and server code
- When converting existing TypeScript interfaces to validated Zod schemas

## What it produces
Zod schema files with TypeScript type inference, validation middleware for API routes and Server Actions, generated schemas via Python automation script, and template files for common patterns (forms, entities, API routes, Server Actions).
