# GitHub Actions CI Workflow

**Category:** automation
**Status:** active
**Version:** v1

## What it does
Sets up comprehensive GitHub Actions CI/CD workflows for modern web applications, covering lint, test, build, and deploy stages with caching strategies and preview deployment URL comments on pull requests.

## When to use
- Configuring a new CI/CD pipeline with GitHub Actions for a web project
- Adding preview deployments that post URL comments to pull requests
- Optimizing existing workflow performance with multi-level caching
- Setting up environment-specific deployment targets (Vercel, Netlify, AWS, etc.)

## What it produces
Generates .github/workflows/ YAML files for CI (lint/test/build), deployment, and preview deployment, along with caching configuration and PR comment automation.
