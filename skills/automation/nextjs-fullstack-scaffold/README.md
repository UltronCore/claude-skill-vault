# Next.js Full-Stack Scaffold

**Category:** misc
**Status:** active
**Version:** v1

## What it does
Scaffolds a production-ready full-stack Next.js 16 application with React 19, TypeScript, Tailwind CSS v4, shadcn/ui components, Supabase authentication, Prisma ORM with PostgreSQL, Zod validation, and a comprehensive testing setup including Vitest and Playwright.

## When to use
- Starting a new full-stack web project and wanting a production-grade boilerplate
- Setting up a Next.js app with auth, database, UI components, and testing in one pass
- Creating a starter template that follows modern Next.js App Router and RSC conventions
- Generating a scaffold with protected routes, server actions, and CI workflow included

## What it produces
Produces a complete project directory with all configuration files, app router pages (auth, protected, dashboard), UI components, Prisma schema, Supabase client setup, server actions with Zod validation, test files, GitHub Actions CI workflow, and a README.
