# Skill Memory: Playwright Flow Recorder

## Purpose
Translates natural language user flow descriptions (plain text, numbered steps, or Given/When/Then format) into executable Playwright TypeScript test scripts. Maps described actions to Playwright API calls using semantic selectors and adds appropriate assertions and waits for reliability.

## Category notes
Belongs in automation because it automates the generation of test scripts from human-readable specifications, bridging requirements documentation and automated test coverage.

## Relationships
Produces test files that integrate with the infrastructure set up by testing-next-stack. Outputs can include accessibility assertions compatible with a11y-checker-ci patterns.
