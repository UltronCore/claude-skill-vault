# ESLint Prettier Husky Config

**Category:** automation
**Status:** active
**Version:** v1

## What it does
Configures comprehensive code quality tooling for Next.js and React projects using ESLint v9 flat config, Prettier formatting, Husky git hooks, lint-staged pre-commit checks, and a GitHub Actions CI lint workflow.

## When to use
- Initializing code quality tooling on a new Next.js or React project
- Migrating from older ESLint config formats to the v9 flat config
- Adding pre-commit hooks to enforce formatting and linting standards
- Setting up CI lint checks on pull requests

## What it produces
Produces eslint.config.mjs, .prettierrc, .prettierignore, Husky pre-commit hook configuration, lint-staged config in package.json, and a GitHub Actions lint workflow file.
