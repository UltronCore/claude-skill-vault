# Review Skills

_Last updated: 2026-03-25_

| Skill | Description | Version | Status |
|-------|-------------|---------|--------|
| [Receiving Code Review](receiving-code-review/) | Establishes a technically rigorous process for evaluating and responding to code review feedback. Requires understanding and verification before implementing any suggestion, pushes back with technical reasoning when feedback is incorrect, and forbids performative agreement or blind implementation. | v1 | ✅ active |
| [Requesting Code Review](requesting-code-review/) | Dispatches a code-reviewer subagent with precisely crafted context (git SHAs, implementation summary, requirements) to catch issues before they compound. The reviewer gets focused context, never the session's conversation history. | v1 | ✅ active |
| [Skill Reviewer and Enhancer](skill-reviewer-and-enhancer/) | Reviews, audits, and enhances existing Claude Code skills to ensure they follow Anthropic best practices, proper structure, current domain-specific patterns, and include all necessary resources. Performs multi-step analysis covering frontmatter validation, instruction style, resource completeness, and domain-specific quality checks. | v1 | ✅ active |
