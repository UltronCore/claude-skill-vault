# Skill Reviewer and Enhancer

**Category:** review
**Status:** active
**Version:** v1

## What it does
Reviews, audits, and enhances existing Claude Code skills to ensure they follow Anthropic best practices, proper structure, current domain-specific patterns, and include all necessary resources. Performs multi-step analysis covering frontmatter validation, instruction style, resource completeness, and domain-specific quality checks.

## When to use
- Auditing an existing skill before deploying it to production
- Updating a skill to follow the latest Anthropic guidelines or framework versions
- Finding and fixing structural issues such as second-person usage or missing resources
- Modernizing outdated skills that reference deprecated APIs or old patterns

## What it produces
A comprehensive skill review report with graded findings, specific improvement recommendations, and optionally automated fixes applied directly to the skill file.
