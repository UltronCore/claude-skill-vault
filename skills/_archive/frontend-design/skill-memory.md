# Skill Memory: Frontend Design

## Purpose
This skill establishes an opinionated approach to frontend creation that prioritizes distinctive aesthetic choices over generic patterns. It requires committing to a bold conceptual direction before writing any code, and explicitly prohibits common "AI slop" aesthetics (Inter/Roboto fonts, purple gradients, predictable layouts). Each implementation should look deliberately designed for its specific context.

## Category notes
Classified as ui-ux because it is focused entirely on frontend interface design and implementation aesthetics.

## Relationships
- brainstorming: typically precedes this skill for complex UI projects
- writing-plans: may be combined when the frontend work is part of a larger plan
