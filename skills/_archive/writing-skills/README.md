# Writing Skills

**Category:** app-usage
**Status:** active
**Version:** v1
**Source:** superpowers@claude-plugins-official

## What it does
Applies TDD principles to skill authoring: run baseline scenarios without the skill first (RED), write minimal skill content addressing the observed failures (GREEN), then close loopholes by finding new rationalizations and adding explicit counters (REFACTOR). Covers YAML frontmatter requirements, Claude Search Optimization, flowchart usage, and token efficiency.

## When to use
- When creating a new skill from scratch
- When editing or improving an existing skill
- When verifying a skill works before deployment

## What it produces
A tested, bulletproof SKILL.md with optimized description, rationalization tables, red flags sections, and no untested changes deployed.
