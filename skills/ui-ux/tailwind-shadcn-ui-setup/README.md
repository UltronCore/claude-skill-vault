# Tailwind + shadcn/ui Setup for Next.js

**Category:** ui-ux
**Status:** active
**Version:** v1

## What it does
Configures a production-ready Tailwind CSS (v3/v4 compatible) and shadcn/ui setup for Next.js 16 App Router projects, including dark mode, design tokens, base layout, and accessibility defaults.

## When to use
- Initializing a new Next.js project with a complete design system
- Adding dark mode support with next-themes
- Setting up a responsive base layout with optional sidebar navigation
- Establishing a semantic CSS variable token system for consistent theming

## What it produces
Configured tailwind.config.ts, global CSS with design tokens, ThemeProvider and mode toggle components, optional AppShell layout, and example pages demonstrating forms, dialogs, and theme customization.
