# Frontend Design

**Category:** ui-ux
**Status:** active
**Version:** v1
**Source:** frontend-design@claude-plugins-official

## What it does
Guides creation of distinctive, production-grade frontend interfaces by first committing to a bold aesthetic direction, then implementing working code with exceptional attention to typography, color, motion, spatial composition, and visual details — deliberately avoiding generic AI aesthetics.

## When to use
- When building any web component, page, or application UI
- When the user asks for a frontend interface of any kind
- When visual quality and memorability matter

## What it produces
Working production-grade HTML/CSS/JS, React, Vue, or other frontend code with a clear, intentional aesthetic point-of-view that avoids generic patterns.

brainstorming, writing-plans
