# uni-app Frontend Development

**Category:** ui-ux
**Status:** active
**Version:** v1

## What it does
Provides a comprehensive guide and reusable templates for building cross-platform apps with uni-app, Vue 3, and TypeScript, covering UI library integration, SCSS theming, state management, and multi-platform build configuration.

## When to use
- Creating new uni-app pages or components for WeChat Mini Programs, H5, or other targets
- Integrating UI libraries such as uView Plus, uni-ui, or TuniaoUI into a uni-app project
- Implementing multi-theme design systems with SCSS variables
- Setting up Pinia state management, API modules, and platform-conditional code

## What it produces
Page components following the standard uni-app CLI project structure, SCSS theme configuration, Pinia store definitions, API request utilities, and platform-specific build configurations.
