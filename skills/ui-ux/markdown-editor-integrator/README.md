# Markdown Editor Integrator

**Category:** ui-ux
**Status:** active
**Version:** v1

## What it does
Installs and configures @uiw/react-md-editor with theme integration, server-side sanitization, controlled/uncontrolled modes, and proper persistence for content-heavy applications.

## When to use
- Adding markdown editing capability to forms or content management features
- Creating rich text editing for entity descriptions, bios, or lore entries
- Implementing WYSIWYG editing with live markdown preview
- Setting up auto-save drafts or local storage persistence for editors

## What it produces
A configured MarkdownEditor wrapper component, a MarkdownPreview display component, theme integration with shadcn/ui, and server-side sanitization utilities.

## Related skills
- form-generator-rhf-zod (for integrating markdown fields into RHF forms)
- tailwind-shadcn-ui-setup (for consistent theme integration)
