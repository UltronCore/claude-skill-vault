# Form Generator with React Hook Form & Zod

**Category:** ui-ux
**Status:** active
**Version:** v1

## What it does
Generates production-ready React forms using React Hook Form, Zod validation schemas, and accessible shadcn/ui form controls. Creates forms with client-side and server-side validation, proper TypeScript types, and consistent error handling.

## When to use
- Creating forms for entities with complex field types and validation requirements
- Building data entry interfaces that need both client and server validation
- Generating accessible forms with proper ARIA attributes
- Setting up forms with multi-step, conditional fields, or array field patterns

## What it produces
A complete form component with Zod schema, zodResolver integration, shadcn/ui FormField structure, server action for submission, and TypeScript types inferred from the schema.
