# UI Library Usage Auditor

**Category:** ui-ux
**Status:** active
**Version:** v1

## What it does
Reviews and audits shadcn/ui component usage across a codebase to ensure accessible, consistent, and maintainable UI patterns, identifying issues and recommending component extractions or layout optimizations.

## When to use
- Auditing UI components for WCAG accessibility compliance
- Identifying duplicate component patterns ready for extraction
- Reviewing shadcn/ui usage consistency across pages
- Checking for proper ARIA attributes, semantic HTML, and keyboard navigation

## What it produces
A structured audit report with critical issues, warnings, and suggestions categorized by accessibility score, component consistency, extraction opportunities, and layout/responsiveness, including file paths and concrete fix examples.
